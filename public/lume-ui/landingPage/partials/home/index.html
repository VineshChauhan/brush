Lume provide amazing features, below are just to name a few:

  • Better and expandable code structure.
  • RESTFul api: using Lumen (Laravel's micro framework)
  • Database migration script and seeds (just execute php artisan migrate)
  • Langing Page UI (Using Bootstrap SASS)
  • Admin Dashboard UI (Using Bootstrap SASS)
  • Automated task using gulp (just execute gulp to compile JS, CSS and HTML files)
  • Automated file refresh for development (Just execute gulp watch development mode)
  • Simple build process (just execute NODE_ENV=prod gulp serve and get the zip file in target folder, ready to ship)

So let's get started and build an amazong app with best practices and best code management.
